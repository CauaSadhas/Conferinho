COMO SUBIR NO GITHUB:
1. Abra o repositório verdadeiro do Conferinho.
2. Clique em Add file > Upload files.
3. Arraste os 3 arquivos acima para a raiz do repositório.
4. Quando o GitHub perguntar, confirme Replace files / substituir arquivos existentes.
5. NÃO apague a pasta assets. O novo index.html continua usando:
   assets/mascote-conferinho.jpg
   assets/favicon.svg
6. Clique em Commit changes.
7. A Vercel deverá publicar a atualização automaticamente.

ATUALIZAÇÃO DA COMPARAÇÃO DE PDF
- A ordem das notas nos relatórios não interfere mais.
- A comparação é feita pelo número da nota.
- Encontrou o mesmo número e o mesmo valor: Conferido.
- Encontrou o mesmo número com valor diferente: Valor divergente.
- Não encontrou o número no outro relatório: Não localizado.
- O leitor possui tratamento específico para relatório SEFAZ/MS, Acompanhamento de Entradas e Livro Caixa.

ATUALIZAÇÃO — REGRA EXATA NF + VALOR
- Exemplo: encontrou a NF 3249028 no Relatório 1.
- O Conferinho identifica o valor ligado a essa NF.
- Depois procura a NF 3249028 em qualquer posição do Relatório 2.
- Mesmo número + mesmo valor: CONFERIDO.
- Mesmo número + valor diferente: DIVERGENTE, mostrando os dois valores.
- Número não encontrado: AUSENTE, informando em qual relatório a NF não está.
- A ordem, página ou posição da nota não interfere na comparação.
- Foram adicionados tratamentos para expressões como NF, NF-e, NFS-e, Nota Fiscal, Número da Nota, Valor da Nota, Valor Contábil, Valor Total e Total NF.
