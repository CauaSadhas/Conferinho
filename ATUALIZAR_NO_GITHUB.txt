COMO ATUALIZAR O CONFERINHO V25 NO GITHUB / VERCEL

1. Faça backup do repositório atual.
2. Abra a pasta conferinho_v25 deste ZIP.
3. Envie TODOS os arquivos de dentro dela para a raiz do repositório.
4. Confirme a substituição dos arquivos existentes.
5. Faça o commit no GitHub.
6. Aguarde a Vercel mostrar READY.
7. Abra o site e pressione Ctrl + F5.

- conciliador.html
- conciliador.css
- conciliador.js

Os módulos de Entradas, NFS-e e Somador foram preservados.
