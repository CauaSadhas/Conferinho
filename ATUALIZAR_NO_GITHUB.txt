COMO SUBIR NO GITHUB:
1. Abra o repositório verdadeiro do Conferinho.
2. Clique em Add file > Upload files.
3. Arraste os 3 arquivos acima para a raiz do repositório.
4. Quando o GitHub perguntar, confirme Replace files / substituir arquivos existentes.
5. NÃO apague a pasta assets. O novo index.html continua usando:
   assets/mascote-conferinho.jpg
   assets/favicon.svg
6. Clique em Commit changes.
7. A Vercel deverá publicar a atualização automaticamente.

A leitura reconhece campos comuns dos padrões ABRASF e NFS-e Nacional. Prefeituras podem usar nomes de campos próprios; nesses casos, basta enviar um XML de exemplo para mapear o layout adicional.
